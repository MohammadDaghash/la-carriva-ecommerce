# La Carriva – E-Commerce Storefront

This repository contains an archived version of **La Carriva**, an e-commerce storefront originally built and deployed on Shopify.

The project is preserved as a portfolio example to demonstrate real-world storefront architecture, data modeling, and theme customization.

## Repository Structure

- `frontend/` – Shopify Online Store 2.0 theme source code
- `data/` – Product data exports used to document the data model
- `demo/` – Screenshots of the live storefront (if applicable)
- `architecture/` – System design notes and decisions (if applicable)

## Status

This repository is **not** a live store.
Checkout and payments are disabled.
Content is provided for architectural and code review purposes only.
